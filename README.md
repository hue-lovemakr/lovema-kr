# Seoul Theme for VS Code

**Seoul Theme** is a premium color suite for VS Code, deeply inspired by the traditional colors of Korea (O-bang-saek and Seoul Colors). It is designed to provide **maximum visual comfort** and **aesthetic satisfaction** through three distinct moods.
